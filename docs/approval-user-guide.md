# Guía del aprobador

Abre **Aprobaciones**, filtra la bandeja y revisa antecedentes, respaldos y workflow congelado. Solo aparecen etapas disponibles según orden, paralelismo, rol, sustitución y unidad. Aprobar, rechazar o solicitar corrección usa el motor PostgreSQL. El comentario y respaldo adicional se exigen cuando la etapa lo configura.
