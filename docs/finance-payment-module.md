# Finanzas → Solicitud de Pagos

El módulo cubre solicitud, workflow configurable, aprobación, corrección, programación, ejecución, comprobantes, dashboard y reportes. PostgreSQL concentra correlativos, snapshots, decisiones, transiciones e idempotencia. RLS limita empresa y unidad; Storage conserva respaldos y comprobantes privados.

No incluye Caja Chica, pagos parciales, integración bancaria ni conciliación automática.
